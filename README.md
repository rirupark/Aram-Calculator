# Aram-Calculator 📆
> Chuseok Challenge 2022 ( 09.09 ~ 10.24 )
### 1. 목적
종강까지 아람(기숙사 학식)을 다 먹기 위해 매일 먹어야 할 식수를 계산해주는 앱
### 2. 기능
- 매일 먹어야 할 식수 계산
- 남은 식수의 비용으로 먹을 수 있는 대체 음식(커피, 국밥, 치킨) 개수 계산
